# Changelog

## [1.0.0] - 2025-02-18

### 首次正式發布

#### 課程管理
- 首頁熱門課程、分類篩選
- 探索課程頁：搜尋、分類、難度篩選
- 建立課程：完整表單、章節與單元、影片連結（YouTube / Google Drive）
- 編輯課程：課程詳情頁編輯入口，可修改章節與單元

#### AI 功能
- 課程 AI 助教：Gemini 驅動即時問答
- AI 協助撰寫課程描述
- 後端 API 代理，保護 Gemini API Key

#### 封面與分類
- 封面編輯器：自訂首頁 Hero 區塊（標籤、主標題、描述、按鈕）
- 分類管理：新增、修改、刪除，修改時同步更新所有課程

#### 介面與體驗
- 深色／淺色模式，偏好儲存
- 響應式設計

#### 部署
- 支援 Zeabur 一鍵部署
- Express 整合前後端，單一服務
